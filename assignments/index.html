Chromebook Accessibility Features Certificate

This certificate details the many different accessibility features that Chromebooks come with preinstalled and how to use many of them. One of the cool things I found out is that Chromebooks can do Braille text for blind or low vision students. I think Chromebooks are a great classroom resource and look forward to potentially incorporating them into my music classroom.

Microsoft module on The Inclusive Educator

This module takes you through what it means to be an inclusive educator and what things you might face as an inclusive educator. I will strive to make my classes as inclusive as possible for all of my students, with or without special needs.

Microsoft module on the Introduction to Inclusive Digital Literacy

This module teaches you what digital literacy is and why it is important for our students to be digitally literate. Using the information I learned in this module, I will strive to make sure my students are digitally literate with all of the relevent musical and assistive technolgies that we have access to.

Microsoft module on Developing a Digitally Literate Learner

This module teaches you what exactly having a digitally literate student means - being able to use a wide variety of tech interchangeably. In my classroom, this will take place by integrating tech into my lessons when possible and making sure students have the basic skills they need to be successful in a digital world.


View

Microsoft module on Developing a Digitally Literate Pedagogy

This module teaches you how to effectively use technology as a teaching tool for your students. In my classroom, this can look like a bunch of different things - maybe using the Microsoft Immersive Reader when working with younger children in a general music classroom, or using a word highlighting tool to track with the lyrics of a song.


View

Microsoft module on Creating a Digitally Literate Curriculum

This module dives into how teachers can build their curriculum with the use of building and increasing technological literacy. In my future music classroom, this would look like doing lessons to make sure kids know how to do things like save files, send emails, use browser extensions and tools, and more.


View

Microsoft module on Assistive Technology

This module talks about all of the assistive technologies that you can think of - multitouch tablets, accessible switches, sensory learning rooms, and more. In my classroom, I will strive to make sure that students have access to the tools they need - be it an iPad to make music with through GarageBand or a tablet to make communication so much easier.
